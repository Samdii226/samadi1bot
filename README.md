# 🤖 Samadi Bot - Multi-Purpose Telegram Bot

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/template/your-template)

A powerful Telegram bot with 6 essential tools built with Python and python-telegram-bot.

## 🚀 Features

| Tool | Description |
|------|-------------|
| 🔗 URL Shortener | Shorten long URLs instantly |
| 📊 Word Counter | Analyze text: words, characters, sentences, paragraphs |
| 🎨 AI Image Generation | Generate images from text prompts |
| 🔄 Image Converter | Convert between JPEG, PNG, WEBP |
| 📐 Image Resizer | Resize images to custom dimensions |
| 📱 QR Code Generator | Generate QR codes from text or URLs |

## 🛠️ Tech Stack

- **Python 3.11+**
- **python-telegram-bot v20.7**
- **Pillow** - Image processing
- **aiohttp** - Async HTTP requests
- **qrcode** - QR generation
- **Railway** - Hosting platform
- **GitHub** - Version control
